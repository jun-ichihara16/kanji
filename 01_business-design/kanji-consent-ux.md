### 画面タイトル

```
ポイント・報酬プログラムについて
```

### 本文

```
KANJIでは、飲み会後の口コミ投稿や
会場への送客実績に応じてポイントをお渡しします。

【ポイントの貯め方】
・公式LINE登録          +10pt
・口コミ投稿（必須4項目）  +15pt
・口コミテキスト投稿      +10pt

【ポイントの使い方】
貯まったポイントはAmazonギフト券や旅行券と交換できます。
（100pt〜交換可能）

【紹介報酬について】
認定幹事（50pt以上）になると、
当社加盟店へのイベント送客実績に応じて
PayPayで報酬をお受け取りいただけます。
（飲食代金5万円以上の場合に適用）

---

⚠️ 重要なご説明

・ポイントは当社が無償で付与するものであり、
  現金との交換はできません。
・ポイントの有効期限は付与日から6ヶ月です。
・紹介報酬は「紹介謝礼」として支払われます。
  年間受取額によっては確定申告が必要な場合があります。
  詳細は利用規約第8条をご確認ください。
・ポイントおよび報酬の内容は変更される場合があります。
```

## UI実装上の注意点

| ポイント | 内容 |
|---|---|
| **チェックボックスは個別に** | 「まとめて同意」ではなく、重要事項は個別チェックを求める |
| **CTAは同意完了まで非活性** | 全チェックが入るまでボタンをグレーアウト |
| **規約はモーダルで表示** | ページ遷移させず、その場で読めるようにする |
| **スキップは小さく** | 公式LINE登録・ポイント参加はスキップ可能だが、目立たせない |
| **ポイント獲得を可視化** | 「+10pt」など、登録時の報酬を具体的に表示してモチベーションを上げる |
| **税務の注意書きは必ず表示** | STEP5の⚠️部分は省略しない（法的保護のため） |
