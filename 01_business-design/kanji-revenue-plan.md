## プラン① 成果報酬プラン

### 概要

KANJI経由で幹事が予約・来店した飲み会に対して、売上の15%を事後精算で受け取る。

### 条件

| 項目 | 内容 |
|---|---|
| **手数料率** | 売上の15%（幹事バック10% ＋ KANJI収益5%） |
| **精算方式** | 都度精算（飲み会ごとに請求・入金） |
| **最低取引金額** | 1回あたり50,000円以上の飲食代が対象 |
| **証憑** | 幹事がレシート画像をLINEで送付 → KANJI確認後に請求 |
| **支払い方法（店舗→KANJI）** | 銀行振込（請求書発行後7営業日以内） |
| **キャッシュバック方法（KANJI→幹事）** | PayPay送金（店舗入金確認後5営業日以内） |

### お金の流れ

```
飲食代 50,000円 の場合

  飲食店 → KANJI：7,500円（15%）を振込
  KANJI → 幹事：5,000円（10%）をPayPay送金
  KANJIの収益：2,500円（5%）
```

### フロー

```
① 幹事がKANJI経由で飲食店を予約
② 飲み会実施
③ 翌日中に幹事がレシート画像をLINEで送付
④ KANJIがレシートを確認（金額・店舗・日付）
⑤ 飲食店に請求書を送付
⑥ 飲食店が7営業日以内に振込
⑦ KANJI確認後、幹事へPayPay送金（5営業日以内）
```

### 注意事項・リスク対策

- レシートの偽造・金額水増しを防ぐため、KANJIが目視確認を実施
- 異常な金額（前回比200%超など）はフラグを立てて保留
- 加盟店は事前に銀行口座情報を登録（未払いリスク低減）
- 最低取引金額50,000円未満の飲み会は対象外（管理コスト削減）

## プラン② LINEリストPRプラン

### 概要

KANJIの公式LINEに登録している幹事リストに対して、飲食店・企業が告知メッセージを配信できるプラン。

### 条件

| 項目 | 内容 |
|---|---|
| **単価** | 1人あたり250円（税別） |
| **最低配信人数** | 50人〜（最低12,500円〜） |
| **配信頻度制限** | 同一店舗・企業から同一ユーザーへの配信は月1回まで |
| **告知文面** | KANJI側で作成（店舗から素材・情報を受け取り制作） |
| **配信対象** | KANJIの公式LINE登録済み幹事リスト |
| **支払い方法** | 配信前払い（請求書発行後に振込確認後に配信） |

### 料金例

| 配信人数 | 費用（税別） |
|---|---|
| 50人 | 12,500円 |
| 100人 | 25,000円 |
| 500人 | 125,000円 |
| 1,000人 | 250,000円 |
| 2,000人 | 500,000円 |

### 配信可能なセグメント（将来的に）

リストが拡大した際に、以下のセグメント指定オプションを追加予定。

| セグメント | 内容 |
|---|---|
| エリア指定 | 渋谷・新宿・六本木など |
| イベント規模 | 10人以上の幹事のみ など |
| 直近活動 | 過去30日以内にイベント作成した幹事 |

### 告知文面の制作フロー

```
① 店舗から素材（店名・写真・訴求内容・URL）を受け取る
② KANJI側でLINEメッセージ文面を作成
③ 店舗が確認・承認
④ 配信実施
```

## プラン③ おすすめ店舗掲載プラン

### 概要

KANJIアプリ内の「おすすめ店舗リスト」ページに飲食店を掲載するプラン。幹事がイベント作成時や会場選びの際に参照するページに常時表示される。

### 条件（案）

| 項目 | 内容 |
|---|---|
| **掲載料** | 月額 10,000〜30,000円（エリア・掲載位置による） |
| **掲載内容** | 店名・写真・特徴・収容人数・予算目安・予約リンク |
| **掲載期間** | 最低1ヶ月〜 |
| **支払い方法** | 月額前払い |

### ページ設計イメージ

```
【おすすめ店舗リスト】

エリア：渋谷 / 新宿 / 六本木 / 銀座 / 全国

┌─────────────────────────────┐
│ [店舗写真]  ○○居酒屋 渋谷店          │
│            収容：〜50名 / 予算：4,000円〜 │
│            「貸し切りOK・幹事特典あり」  │
│            [予約する →]               │
└─────────────────────────────┘
```

### 成果報酬プランとの組み合わせ

掲載プランに加入している店舗が成果報酬プランも契約している場合、掲載ページからの予約は成果報酬の対象となる。掲載料 ＋ 成果報酬の二重収益が発生する。
