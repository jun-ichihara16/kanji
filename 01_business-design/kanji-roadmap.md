# KANJI 事業全体ロードマップ

**更新日：** 2026年4月2日
**ステータス：** Phase 1 進行中

---

## コンセプト

KANJIは **「割り勘ツール → 連絡ツール → 集客プラットフォーム」** の3段階で進化する。

まず **割り勘機能** で幹事の最大ペインを解消し、ユーザー基盤を獲得。
次に **連絡ツール**（招待・リマインド・ヒアリング）で幹事のイベント運営を丸ごとカバー。
最終的に、飲食店・施設からの **集客クエスト（案件）** を幹事が受け取り、
採用イベント・フェス・パーティの集客を担う **幹事プラットフォーム** へ進化する。

## Phase 1｜割り勘MVP（2026年4月〜6月）

**テーマ：「集金のしんどさ」をゼロにする**

### 目的
割り勘・PayPay番号収集・集金管理の体験を磨き切り、「これだけで使う価値がある」状態を作る。

### 収益
なし（完全無料）

### 実装する機能

| 優先度 | 機能 | 内容 |
|--------|------|------|
| ★★★ | LINEログイン・イベント作成 | 幹事がLINEで即スタート |
| ★★★ | PayPay番号収集 | 参加者がURLを踏んで30秒で登録 |
| ★★★ | 割り勘計算 | 端数処理・傾斜配分に対応 |
| ★★★ | 集金管理・未払いリマインド | 支払い状況の可視化と自動催促 |
| ★★☆ | 参加者リスト自動蓄積 | 友達追加を検知してリストに追加 |

### 重点KPI

| KPI | 目標値 |
|-----|--------|
| 月間イベント作成数 | 30件/月 |
| 2回目イベント作成率 | 40%以上 |
| 参加者PayPay登録完了率 | 70%以上 |
| NPS（推奨度） | +20以上 |

### Phase 2への移行基準
1. 月間イベント作成数 **30件/月を2ヶ月連続** で達成
2. 2回目イベント作成率 **40%以上**

## Phase 2｜連絡ツール拡張（2026年7月〜9月）

**テーマ：「幹事のイベント運営」を丸ごとカバーする**

### 目的
割り勘だけでなく、招待・リマインド・振り返りまでイベントの連絡業務を一括で担い、幹事にとって「これがないと困る」ツールになる。

### 収益
なし（ユーザー基盤拡大フェーズ）

### 実装する機能

| 優先度 | 機能 | 内容 |
|--------|------|------|
| ★★★ | 招待メッセージ一斉送信 | 蓄積リストからチェック→AIが文面生成→LINE送信 |
| ★★★ | イベントリマインド自動化 | 前日・当日の自動通知（集合場所・時間） |
| ★★☆ | 参加者への質問自動返答 | 「場所どこ？」「何時？」にLINEで自動回答 |
| ★★☆ | 振り返りヒアリング | イベント後にKANJIが会話形式でフィードバック収集 |
| ★☆☆ | イベント履歴・アーカイブ | 過去の飲み会一覧・参加者・金額を閲覧 |

### 重点KPI

| KPI | 目標値 |
|-----|--------|
| 月間イベント作成数 | 100件/月 |
| 招待メッセージ送信利用率 | 30%以上 |
| 幹事登録者数 | 300人 |
| LINE友達追加率（参加者） | 50%以上 |

### Phase 3への移行基準
1. 幹事登録者数 **300人以上**
2. 月間イベント作成数 **100件/月**

## Phase 3｜幹事特典・提携店舗（2026年10月〜12月）

**テーマ：「幹事が得をする」体験を作り、収益化を開始する**

### 目的
提携店舗・施設を獲得し、KANJI経由予約の成果報酬で収益化開始。幹事ポイント制度を導入してリテンションを強化する。

### 収益
成果報酬 5%（KANJI取り分）

### 実装する機能

| 優先度 | 機能 | 内容 |
|--------|------|------|
| ★★★ | 提携店舗ページ | おすすめ店舗一覧・KANJI経由予約 |
| ★★★ | 幹事ポイント管理 | ポイント残高・履歴・特典交換UI |
| ★★★ | 特典交換フロー | サウナ・グランピング・ヴィラの予約申請 |
| ★★☆ | レシート投稿 | 飲み会後の実績記録・ポイント付与 |
| ★☆☆ | 口コミ投稿 | 店舗レビュー（認定幹事への道） |

### 重点KPI

| KPI | 目標値 |
|-----|--------|
| 提携店舗・施設数 | 10店舗以上 |
| 月間KANJI経由予約数 | 50件/月 |
| 月次収益（KANJI取り分） | 25万円/月 |
| 幹事登録者数 | 500人 |

### Phase 4への移行基準
1. 月次収益 **25万円/月を2ヶ月連続** で達成
2. 提携店舗数 **10店舗以上**

## Phase 4｜集客プラットフォーム化（2027年1月〜）

**テーマ：「幹事リスト」を集客インフラに変える**

### 目的
飲食店・施設・企業から「集客クエスト（案件）」を受け、幹事がイベントを開催して集客を実現する。幹事にインセンティブが働く仕組みで、自律的にイベントが増え続けるプラットフォームへ進化する。

### 収益
成果報酬 ＋ 集客クエスト手数料 ＋ LINEリストPRプラン ＋ 店舗掲載プラン

### ユースケース

| クエスト種別 | 発注者 | 幹事がやること | 幹事報酬 |
|-------------|--------|---------------|---------|
| 飲食店の集客 | 飲食店オーナー | 「あの店で飲み会しよう」と企画・集客 | ポイント or 割引 |
| 採用イベント | 企業の人事 | 「うちの会社のカジュアル飲み会」を企画・集客 | 報酬＋特典 |
| フェス・パーティ | イベント主催者 | 友人グループ単位で集客・チケット販売 | インセンティブ |
| 施設体験会 | サウナ・ヴィラ等 | 「サウナ行かない？」と企画・集客 | 無料招待枠 |

### 実装する機能

| 優先度 | 機能 | 内容 |
|--------|------|------|
| ★★★ | 集客クエスト一覧 | 幹事が受けられる案件の一覧・詳細・応募 |
| ★★★ | クエスト管理ダッシュボード | 応募→集客→完了→報酬確認の一気通貫UI |
| ★★★ | LINEリストPR配信 | 幹事リストへの店舗・施設・イベントPR配信 |
| ★★☆ | 認定幹事ランクシステム | 実績に応じたランク・案件の優先配信 |
| ★★☆ | 法人向け管理画面 | 企業・店舗がクエストを登録・管理するダッシュボード |
| ★☆☆ | 法人向け経費精算連携 | 社内イベントの経費精算書自動生成 |

### MRR 100万円の内訳シミュレーション

| 収益源 | 条件 | 月次収益 |
|--------|------|---------|
| 成果報酬（5%） | 月200件 × 平均2,500円 | 50万円 |
| 集客クエスト手数料 | 月30件 × 平均5,000円 | 15万円 |
| LINEリストPRプラン | 2,000人 × 250円 × 月1配信 | 50万円 |
| **合計** | | **115万円** |
