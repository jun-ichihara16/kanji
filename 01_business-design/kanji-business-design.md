## 1. ビジネスモデルの核心

### お金の流れ（シンプル版）

```
飲食店の売上
    ↓ 15%を事後送金
  KANJI
    ↓ 10%を幹事へバック
  幹事（LINE公式アカウント登録済み）
    ↓ 残り5%がKANJIの収益
  合同会社CUEN
```

| 関係者 | 役割 | お金の動き |
|---|---|---|
| **飲食店** | 集客コストとして15%を支払う | 売上の15%を事後送金（レシート確認後） |
| **KANJI** | 送客プラットフォーム＋精算管理 | 15%受取 → 10%を幹事へ送金、5%を保持 |
| **幹事** | 予約者・送客者 | 飲み会後に売上の10%をキャッシュバック |

### 具体例

> 10人の飲み会、一人5,000円 = 合計50,000円の飲食代
>
> - 飲食店 → KANJI：7,500円（15%）を送金
> - KANJI → 幹事：5,000円（10%）をLINE Pay / 振込でバック
> - KANJIの収益：2,500円（5%）

## 2. なぜ幹事が公式LINEに登録する理由ができるか

これが**このモデルの最も重要な設計**である。

幹事が公式LINEに登録する理由：
1. **キャッシュバックを受け取るため**（振込先の登録が必要）
2. **次回の飲み会でもKANJIを使いたいから**（体験が良い）
3. **おすすめ店舗情報が届く**（自分が得する情報）

→ 「登録しないと損」という設計になっているため、自然にリスト化が進む。

## 3. キャッシュフロー設計（健全化のポイント）

### 問題：先払いリスクをどう排除するか

悪い設計：KANJIが先に幹事へ払い、後で店舗から回収 → 未回収リスク発生

**良い設計（採用案）：**

```
① 幹事が事前予約（KANJI経由）
       ↓
② 飲み会実施
       ↓
③ 幹事がレシート画像をLINEで送付（翌日以内）
       ↓
④ KANJIがレシートを確認（AI自動認識 or 手動）
       ↓
⑤ 店舗に請求書を送付（月次 or 都度）
       ↓
⑥ 店舗が15%を送金（支払いサイト：翌月末など）
       ↓
⑦ KANJIが幹事へ10%をバック（店舗入金確認後）
```

**キャッシュフローのポイント：**
- KANJIは**店舗から入金を確認してから**幹事へバック → 立替えリスクゼロ
- 支払いサイトを「翌月末払い」にすることで、KANJIは一時的に資金を保持できる
- 幹事へのバックは「店舗入金後5営業日以内」と明示することで信頼を担保

### キャッシュフロー図

```
月初                 月中                 月末
  |                    |                    |
  | 飲み会実施          | 店舗へ請求書送付    | 店舗から15%入金
  | レシート受取        |                    |   ↓
  |                    |                    | 幹事へ10%バック
  |                    |                    | 5%がKANJI収益
```

```

  公式LINE登録
      ↓

      ↓
```

## 5. 加盟店開拓の設計

### 飲食店が加盟するメリット

- **集客コストが完全成果報酬**：来客がなければ費用ゼロ
- **幹事という意思決定者に直接リーチ**できる
- **予約の確実性が高い**：KANJI経由は「飲み会を計画中の幹事」なので来店率が高い

### 飲食店の既存集客コストとの比較

| 媒体 | コスト構造 | 実質コスト |
|---|---|---|
| ホットペッパー | 月額掲載料（数万〜数十万円） | 来客ゼロでも発生 |
| 食べログ広告 | 月額固定 | 同上 |
| **KANJI** | **売上の15%（完全成果報酬）** | **来客があった時だけ** |

→ 飲食店にとってリスクゼロの集客手段として訴求できる。

### 初期加盟店の開拓戦略

1. **渋谷・新宿・六本木の居酒屋チェーン**から営業開始（幹事需要が最も高いエリア）
2. **「最初の3ヶ月は手数料10%に割引」**などの初期インセンティブで導入ハードルを下げる
3. 加盟店が増えるほど幹事にとってのKANJIの価値が上がる（ネットワーク効果）

## 6. MRR成長シミュレーション

### 前提条件

- 平均飲み会単価：50,000円（10人 × 5,000円）
- KANJI取り分：5%（2,500円/件）
- 月間取扱件数が収益を決める

| 月間件数 | 月次収益（5%） | 幹事バック（10%） | 店舗負担（15%） |
|---|---|---|---|
| 100件 | 25万円 | 50万円 | 75万円 |
| 200件 | 50万円 | 100万円 | 150万円 |
| 400件 | 100万円 | 200万円 | 300万円 |

**MRR 100万円 = 月400件の飲み会送客**

月400件は、加盟店20店舗が月20件ずつ予約を受ければ達成できる。

## 7. リスクと対策

| リスク | 内容 | 対策 |
|---|---|---|
| **レシート偽造** | 幹事が金額を水増しして申請 | AI+目視でレシート確認。異常値はフラグ立て |
| **店舗の未払い** | 加盟店が15%を払わない | 事前に銀行口座情報を登録。月次自動引き落とし設計 |
| **幹事の離脱** | キャッシュバックより手間が勝る | レシート送付をLINEで完結（写真1枚送るだけ）にして摩擦を最小化 |
| **競合参入** | 大手が同様のモデルを展開 | 幹事リストと加盟店ネットワークの先行優位。LINEという日本固有の資産 |
