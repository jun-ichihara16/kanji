# ClaudeCode プロンプト — KANJI サービスイメージサイト（デモ画面）

---

以下の仕様に従い、GitHub Pagesにそのままデプロイできる **単一HTMLファイル（app.html）** を生成してください。
これはLPの「無料で始める」ボタンから遷移するサービスのイメージサイトです。
**実際に動くアプリのUIをモックアップとして見せる**ことが目的です。

---

## 概要

**サービス名：** KANJI（幹事アプリ）
**このページの目的：** 「実際に使ったらこんな感じ」というUIイメージを見せて、ユーザーの期待値を高める
**ファイル形式：** 単一HTML（外部ファイル不要。CSS・JSはすべてインライン）
**ホスティング：** GitHub Pages（静的配信）

---

## デザイン仕様

- **カラー：** メイン `#22C55E`（グリーン）、背景 `#FFFFFF`、テキスト `#1A1A1A`、サブテキスト `#6B7280`、ライトグリーン背景 `#F0FDF4`、ボーダー `#E5E7EB`
- **フォント：** Google Fonts CDNから `Noto Sans JP`（本文）と `Inter`（数字・英字）を読み込む
- **スタイル：** スマホアプリのUIをブラウザで再現したデザイン。LINEアプリのような親しみやすさ。
- **画像：** 一切使用しない。アイコンはすべてUnicode絵文字またはインラインSVGで代替
- **レスポンシブ：** スマホファースト。PC表示時はスマホ画面（max-width: 390px）をセンタリングして表示し、周囲にグレー背景を敷く
- **アニメーション：** 画面遷移時のスライドアニメーション（左右スライド）を実装

---

## ページ全体の構造

このページは **タブ切り替え式の擬似アプリ画面** として実装する。
上部に「幹事視点」「参加者視点」の2つのタブを設け、タブをクリックすると対応するデモ画面に切り替わる。

```
[ヘッダー]
  KANJI ロゴ（左）  ← LPに戻るリンク（右）

[タブ切り替え]
  [幹事として使う]  [参加者として使う]

[デモ画面エリア]
  ← タブに応じた画面が表示される →

[フッター]
  「実際に使ってみる（LINEログイン）」ボタン
  ※ ボタンのリンク先は # （仮）
```

## インタラクション仕様

### タブ切り替え
- 「幹事として使う」「参加者として使う」タブをクリックで切り替え
- アクティブタブはグリーン下線 + グリーンテキスト
- 切り替え時にフェードアニメーション

### ステップ切り替え（← →ボタン）
- 各タブ内でSTEP 1〜3を左右スライドで切り替え
- 左上に「← 戻る」、右下に「次へ →」ボタン
- ステップインジケーター（●○○）でどのステップかを表示
- STEP 3の「次へ」は非表示

### PayPay番号一覧の展開
- 幹事STEP 3の「PayPay番号をまとめて見る」ボタンをクリックで展開・折りたたみ

## 技術要件

1. **単一HTMLファイル**（app.html）として出力。CSSとJSはすべて `<style>` と `<script>` タグ内に記述
2. **Google Fonts** を `<link>` タグでCDNから読み込む（Noto Sans JP + Inter）
3. **タブ切り替え** をJSで実装（クリックで幹事/参加者を切り替え）
4. **ステップ切り替え** をJSで実装（← →ボタンでSTEP 1〜3を切り替え）
5. **スライドアニメーション** をCSSで実装（translateX + transition）
6. **PayPay番号の展開/折りたたみ** をJSで実装
7. **PC表示時** はスマホ画面（max-width: 390px）をセンタリング。周囲は `#F3F4F6`（ライトグレー）背景
8. **外部画像・外部CSSフレームワーク（Tailwind等）は一切使用しない**
9. **LPへの戻りリンク** は `index.html` へ
10. **CTAボタンのリンク先** はすべて `#`（仮）
