# KANJI 利用規約

**最終更新日：2026年4月1日（ドラフト版）**

---

## 第1条（目的および適用）

本規約は、KANJI（以下「当サービス」）を運営する株式会社●●（以下「当社」）が提供するイベント幹事支援サービスの利用条件を定めるものです。本サービスを利用するすべてのユーザー（幹事ユーザー・参加者ユーザーを含む）は、本規約に同意したものとみなします。

---

## 第2条（定義）

本規約において使用する用語の定義は以下のとおりです。

| 用語 | 定義 |
|---|---|
| **幹事ユーザー** | 本サービスを通じてイベントを作成・管理するユーザー |
| **参加者ユーザー** | 幹事ユーザーが作成したイベントに参加するユーザー |
| **加盟店** | 当社と紹介手数料契約を締結した飲食店・会場等の事業者 |
| **KANJIポイント** | 幹事ユーザーが所定の活動を通じて無償で取得できる当社独自のポイント |
| **紹介報酬** | 幹事ユーザーが加盟店へのイベント送客を行った対価として当社から支払われる報酬 |
| **PRプラン** | 加盟店が幹事ユーザーリストに対して告知を行う広告サービス |

---

## 第3条（アカウント登録）

1. 本サービスの利用には、LINEアカウントによるログインが必要です。
2. 幹事ユーザーとして本サービスを利用するには、当社の公式LINEアカウントへの友だち追加が必要です。
3. ユーザーは、登録情報を正確かつ最新の状態に保つ義務を負います。
4. 未成年者が本サービスを利用する場合は、法定代理人の同意を得るものとします。
5. 一人のユーザーが複数のアカウントを作成することを禁止します。

---

## 第4条（サービスの内容）

当社は、以下のサービスを提供します。

1. **イベント管理機能**：飲み会・食事会等のイベント作成、参加者管理、費用精算の補助
2. **割り勘計算機能**：参加者ごとの負担額の自動計算
3. **加盟店予約支援**：当社加盟店への予約取次ぎおよび送客
4. **KANJIポイントプログラム**：所定の活動に対するポイント付与（詳細は第7条）
5. **紹介報酬プログラム**：加盟店への送客実績に基づく報酬支払い（詳細は第8条）
6. **おすすめ店舗ページ**：幹事ユーザーの口コミ・実績データに基づく加盟店情報の掲載

---

## 第5条（加盟店との取引）

1. 幹事ユーザーが当社を通じて加盟店を予約・利用した場合、当社は当該加盟店との間で締結した紹介手数料契約に基づき、飲食代金等の売上総額の15%（税抜）を紹介手数料として受領します。
2. 前項の紹介手数料は、幹事ユーザーによるレシートの提出確認後、当社が加盟店に対して請求します。
3. 加盟店との取引における最低取引金額は50,000円（税込）とします。これを下回る取引については、紹介報酬プログラムの対象外となります。
4. 当社と加盟店との間の紹介手数料の精算は、取引ごとの都度精算とします。

---

## 第6条（レシート提出および口コミ投稿）

1. 幹事ユーザーが紹介報酬プログラムおよびKANJIポイントの付与を受けるためには、イベント終了後に以下の情報を提出する必要があります。

   | 項目 | 必須/任意 |
   |---|---|
   | 店舗名 | **必須** |
   | 参加人数 | **必須** |
   | レシートの写真 | **必須** |
   | 店舗写真（1枚以上） | **必須** |
   | 口コミテキスト | 任意 |
   | 総合評価（星1〜5） | 任意 |

2. 提出されたレシートおよび写真は、当社が真正性を確認した後に有効とみなします。
3. 虚偽のレシートや改ざんされた情報を提出した場合、当社はアカウントを停止し、付与済みポイントおよび報酬を無効とすることができます。
4. 提出された口コミ・写真は、当社のおすすめ店舗ページおよびサービス改善目的に使用することがあります。ユーザーは、これらの情報を当社が利用することに同意するものとします。

## 第7条（KANJIポイントプログラム）

### 7-1 ポイントの性質

1. KANJIポイント（以下「ポイント」）は、当社が幹事ユーザーに対して無償で付与する当社独自のポイントです。
2. ポイントは、当社が加盟店に対して提供する紹介・送客サービスの対価として、当社が幹事ユーザーに支払う**紹介謝礼の一形態**です。ポイントは、ユーザーが当社に対して金銭を支払って取得するものではありません。
3. ポイントは、資金決済法（平成21年法律第59号）第3条第1項に定める「前払式支払手段」には該当しません。
4. ポイントは、景品表示法（昭和37年法律第134号）上の「景品類」には該当しないものとして設計されています。ポイントの付与は、当社サービスの利用に対する謝礼・報酬として行われるものであり、顧客誘引を主目的とした景品の提供ではありません。

### 7-2 ポイントの付与

以下の活動に対してポイントを付与します。

| 活動 | 付与ポイント |
|---|---|
| 公式LINE友だち追加 | 10pt |
| イベント完了後の口コミ投稿（必須4項目すべて提出） | 15pt |
| 口コミテキスト投稿 | 10pt |
| 店舗写真追加投稿（2枚目以降、1枚につき） | 5pt |
| 参加人数・予算の詳細記録 | 5pt |

### 7-3 ポイントの有効期限

1. ポイントの有効期限は、付与日から**6ヶ月以内**とします。
2. 有効期限を超過したポイントは自動的に失効し、復元はできません。
3. 有効期限が近づいた場合、当社はLINEメッセージ等で通知するよう努めますが、通知の不達による失効について当社は責任を負いません。

### 7-4 ポイントの交換

1. 所定のポイント数に達したユーザーは、以下の特典と交換することができます。

   | 交換先 | 必要ポイント |
   |---|---|
   | Amazonギフト券（500円相当） | 100pt |
   | Amazonギフト券（1,000円相当） | 200pt |
   | 旅行券（5,000円相当） | 1,000pt |
   | 旅行券（10,000円相当） | 2,000pt |

2. ポイントの交換レートおよび交換先は、当社の判断により変更することがあります。変更の際は、本規約の改定手続きに従い事前に通知します。
3. ポイントは現金への換金はできません。
4. ポイントの譲渡・売買・換金は禁止します。

### 7-5 認定幹事制度

1. 累計ポイントに応じて、以下のランクを付与します。

   | ランク | 累計ポイント | 特典 |
   |---|---|---|
   | 一般幹事 | 0〜49pt | 基本機能のみ |
   | 認定幹事 | 50〜149pt | 紹介報酬案件の先行招待 |
   | プレミアム幹事 | 150pt〜 | 優先案件割り当て・特別報酬率 |

2. ランクの判定は、累計ポイント数に基づき自動的に行われます。

## 第8条（紹介報酬プログラム）

### 8-1 報酬の性質

1. 紹介報酬は、幹事ユーザーが当社を通じて加盟店にイベントを送客した実績に対して、当社が支払う**紹介料・謝礼**です。
2. 紹介報酬は、当社が加盟店から受領した紹介手数料（売上の15%）のうち、**売上の10%相当**を幹事ユーザーに支払うものです。
3. 紹介報酬は、当社の収益から支払われるものであり、ユーザー間の資金移動ではありません。

### 8-2 報酬の支払い条件

1. 以下の条件をすべて満たした場合に、紹介報酬の支払い対象となります。
   - 当社加盟店を通じて予約・利用したイベントであること
   - 飲食代金等の総額が50,000円（税込）以上であること
   - 第6条に定める必須4項目（店舗名・参加人数・レシート・写真）を提出し、当社が真正性を確認したこと
   - 加盟店から当社への紹介手数料（15%）の入金が確認されたこと

2. 当社は、加盟店からの入金確認後、**14営業日以内**に幹事ユーザーへの報酬支払い手続きを行います。

### 8-3 報酬の支払い方法

1. 紹介報酬は、幹事ユーザーが事前に登録したPayPayアカウントへの送金により支払います。
2. PayPayアカウントの登録・変更は、ユーザー設定画面から行うものとします。
3. PayPayアカウントの登録がない場合、報酬の支払いは保留されます。登録後30日以内に支払い手続きを行います。

### 8-4 税務上の取り扱い

1. 紹介報酬は、受取人（幹事ユーザー）にとって**雑所得**に該当する場合があります。
2. 年間の紹介報酬受取額が一定額を超える場合、確定申告が必要になることがあります。
3. 当社は、紹介報酬の支払いに際して源泉徴収を行いません。税務上の申告・納税はユーザー自身の責任において行ってください。
4. 税務上の詳細については、税理士等の専門家にご相談ください。

## 第9条（PRプランおよび広告配信）

1. 当社は、加盟店等の広告主から依頼を受け、幹事ユーザーの公式LINEアカウントに対して広告・告知を配信することがあります（以下「PRプラン」）。
2. PRプランによる配信は、同一ユーザーに対して**月1回**を上限とします。
3. 配信内容は当社が審査・作成し、不適切な内容の配信は行いません。
4. ユーザーは、公式LINEアカウントのブロックにより配信を停止することができます。ただし、ブロックによりKANJIポイントの通知等も受け取れなくなることをご了承ください。

## 第10条（禁止事項）

ユーザーは以下の行為を行ってはなりません。

1. 虚偽のレシート・写真・口コミの提出
2. 架空のイベントを作成してポイントまたは報酬を不正取得する行為
3. 複数アカウントを利用した不正なポイント・報酬の取得
4. ポイントの第三者への譲渡・売買・換金
5. 当社または加盟店の名誉・信用を毀損する行為
6. 本サービスを通じて取得した他のユーザーの個人情報の無断利用
7. 当社のシステムへの不正アクセス・改ざん
8. その他、当社が不適切と判断する行為

## 第11条（サービスの変更・停止・終了）

1. 当社は、事前の通知なくサービスの内容を変更することがあります。
2. 当社は、システムメンテナンス、天災、その他やむを得ない事由によりサービスを一時停止することがあります。
3. 当社がサービスを終了する場合、30日前までにユーザーに通知します。
4. サービス終了時に残存するポイントについては、当社が別途定める方法により対応します。

## 第12条（免責事項）

1. 当社は、本サービスの提供に際して最善を尽くしますが、サービスの完全性・正確性・継続性を保証しません。
2. 加盟店との取引において生じたトラブル（サービスの質、衛生問題等）について、当社は責任を負いません。
3. ユーザーが本規約に違反したことにより生じた損害について、当社は責任を負いません。
4. 当社の責任は、直接かつ通常の損害に限り、逸失利益・間接損害は含みません。

---

## 第13条（個人情報の取り扱い）

1. 当社は、ユーザーの個人情報を別途定めるプライバシーポリシーに従い適切に管理します。
2. 提出されたレシート・写真・口コミは、サービス改善および加盟店情報の提供目的に使用します。
3. 幹事ユーザーのリスト情報（氏名・LINEアカウント等）は、PRプランの配信目的に使用することがあります。第三者への販売・提供は行いません。

---

## 第14条（規約の変更）

1. 当社は、必要に応じて本規約を変更することができます。
2. 重要な変更を行う場合は、変更の30日前までに公式LINEアカウントまたはサービス内通知にて告知します。
3. 変更後も本サービスを継続利用した場合、変更後の規約に同意したものとみなします。
